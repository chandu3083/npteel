# Certificate Files Repository

This repository contains certificate files in different formats for viewing and verification purposes.

## Files Included

- **certificate.html**  
  HTML version of the certificate for browser viewing.

- **certificate2.jpg**  
  Image (JPG) version of the certificate.

- **certificate2.pdf**  
  PDF version of the certificate for download or printing.

## How to Use

- Open `certificate.html` in any web browser to view the certificate.
- Download `certificate2.jpg` to view the certificate as an image.
- Download `certificate2.pdf` for a printable or shareable version.

## Notes

- All files represent the same certificate in different formats.
- Use the format that best suits your needs.
